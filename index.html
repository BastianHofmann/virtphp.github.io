Installing VirtPHP

Run this in your terminal to get the latest VirtPHP version:

$ curl -sS http://virtphp.org/installer | php

Or if you don't have curl:

php -r "eval('?>'.file_get_contents('http://virtphp.org/installer'));"

This installer script will simply check some php.ini settings, warn you if they are set incorrectly, and then download the latest virtphp.phar in the current directory

Installer Options

You can install VirtPHP to a specific directory by using the --install-dir option and providing a target directory. Options must be appended to -- so that PHP ignores them, like -- --install-dir=/usr/local/bin, example:

curl -sS https://virtphp.org/installer | php -- --install-dir=bin

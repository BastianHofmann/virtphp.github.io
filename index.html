VirtPHP

Isolated PHP environments

What Is VirtPHP?

VirtPHP is a tool for creating isolated PHP environments. Think virtualenv for PHP.

Most people have systems with one PHP, one place to install PEAR packages, and one place to install PECL extensions, and those are the versions that must be used for all development on that system. That's not really ideal if you need to do development using various versions of PEAR packages or PECL extensions.

VirtPHP to the rescue!

VirtPHP creates isolated environments so that you may run any number of PHP development projects, all using different versions of PEAR packages and different PECL extensions. You may even specify a different version of PHP, if your system has various installations of PHP (i.e. you've built multiple versions for testing).

Is It Ready? Can I Use It?

No. It's not ready yet.

We developed VirtPHP over the span of 48 hours at Hack Nashville on May 4-5, 2013. So, it's still very new. We haven't even dusted the wood shavings off or sanded down the rough edges yet.

Before VirtPHP is ready to release to the wild and open for pull requests and your contributions, we have some clean-up work to do. So, be patient. You'll be able to try it out soon!
